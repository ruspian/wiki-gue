## Kenapa Harus C++? Kenapa Gak Python Aja? ðŸ˜«

Gue tau, Python itu enak. Bahasanya manusiawi. Tapi kenapa kurikulum kampus maksa pake C++?

1.  **Biar Paham Jeroan:** C++ itu bahasa "Telanjang". Lo jadi tau gimana cara kerja RAM, Processor, dan Memori. Kalau lo bisa C++, bahasa lain itu _Ezet Geming Coy_.
2.  **Disiplin Militer:** Python itu manja (salah dikit dimaafin). C++ itu galak (kurang satu titik koma, satu aplikasi hancur). Ini melatih ketelitian lo.
3.  **Warisan Leluhur:** Bahasa ini udah dipake sejak Bapak lo belum ketemu Ibu lo. Windows, Chrome, Game Engine, semuanya dibangun pake C++.

## Perbedaan JavaScript (JS) vs C++ ðŸ¥Š

Buat lo yang mungkin udah pernah nyicip Web Dev (JS):

| Fitur           | JavaScript (Gen Z)                   | C++ (Ibu-Ibu Tantrum)                                     |
| :-------------- | :----------------------------------- | :-------------------------------------------------------- |
| **Gaya Bahasa** | Santuy, fleksibel, campur-campur.    | Kaku, formal, harus sesuai aturan.                        |
| **Tipe Data**   | Bebas! Variabel bisa diisi apa aja.  | **STRICT!** Wadah nasi gak boleh diisi air.               |
| **Kecepatan**   | Lumayan lah.                         | **Ngebut Parah** (Langsung ngobrol sama mesin).           |
| **Error**       | Sering diem-diem error (di console). | Langsung teriak error sebelum dijalankan (Compile Error). |
