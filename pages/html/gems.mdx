## 1. Accordion (Buka-Tutup) Tanpa JS ðŸ˜²

Mau bikin halaman **FAQ (Tanya Jawab)** yang bisa diklik terus muncul jawabannya?
Gak perlu React State `isOpen` / `setIsOpen`. Pake tag `<details>` dan `<summary>`.

```html
<details>
  <summary>Apakah Pendaftaran Santri Masih Buka?</summary>
  <p>Masih dong! Pendaftaran buka sampai tanggal 30 Desember.</p>
</details>

<details>
  <summary>Berapa Biaya SPP?</summary>
  <p>Gratis buat yatim piatu. Bayar buat yang mampu.</p>
</details>
```

* `<details>`: Wadah utamanya.

* `<summary>`: Judul yang selalu kelihatan (bisa diklik).

* Sisanya bakal tersembunyi dan cuma muncul kalau diklik. Magic! âœ¨

## 4. Meta Tags (Biar Ganteng di WA) ðŸ“±

Pernah share link di WhatsApp terus muncul gambar thumbnail dan judul yang keren? Itu namanya Open Graph (OG Tags).

Kalau di HTML biasa ditaruh di `<head>`. Kalau di Next.js, kita atur di file `layout.tsx` atau `page.tsx` pake metadata.

Contoh Logic-nya:
```html

<title>Pondok Pesantren Modern</title>

<meta name="description" content="Pondok terbaik di desa..." />

<meta property="og:title" content="Pondok Pesantren Modern" />
<meta property="og:description" content="Ayo mondok! Fasilitas lengkap, gratis WiFi." />
<meta property="og:image" content="[https://website-lo.com/foto-cover.jpg](https://website-lo.com/foto-cover.jpg)" />
```
