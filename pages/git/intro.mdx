## 3. Konfigurasi Awal (Wajib) âš™ï¸

Setelah install, lo harus kenalan dulu sama Git. Ini cuma perlu dilakuin **SEKALI SEUMUR HIDUP** di laptop yang sama.

Buka terminal (Git Bash di Windows / Terminal di Mac), ketik:

```bash
# 1. Kasih tau nama lo (Bebas)
git config --global user.name "Otong Keren"

# 2. Kasih tau email lo (WAJIB SAMA dengan akun GitHub lo nanti)
git config --global user.email "email.lo@gmail.com"
```

Cek apakah udah kesimpen:

```bash
git config --list
```
